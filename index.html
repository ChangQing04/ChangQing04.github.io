Redis 基础与应用
发表于2025-09-14|数据库|Redis•数据库•缓存
Redis 基础与应用Redis（Remote Dictionary Server）是一种基于 内存 的高性能键值对（Key-Value）数据库，常用于 缓存、分布式锁、消息队列、排行榜、计数器 等场景。 1. Redis 的特点 高性能：数据存放在内存，读写速度极快（10w+ QPS）。 数据结构丰富：支持 String、List、Hash、Set、ZSet 等。 持久化：支持 RDB 快照和 AOF 日志，保证数据安全。 支持集群：支持主从复制、哨兵模式、分片集群。 多语言支持：几乎所有主流编程语言都支持 Redis 客户端。 2.
